Tantangan: Menyisipkan CSS

Menengah

Instruksi Tugas

Saatnya mempercantik halaman web!

Tambahkan CSS ke dalam HTML untuk mengubah tampilan elemen. Gunakan tag <style> di dalam <head> untuk menyisipkan CSS internal.

Ubah warna, background, dan spacing sesuai dengan contoh yang diberikan.

✅ Yang harus kamu capai:

  • • Gunakan tag <style> di dalam <head>
  • • Ubah warna teks heading menjadi biru
  • • Ubah warna background body menjadi abu-abu muda
  • • Tambahkan margin pada paragraf

💡 Hasil yang Diharapkan:

Selamat Datang

Ini adalah contoh paragraf dengan styling CSS yang telah diterapkan.

Heading berwarna biru, background abu-abu muda, dan paragraf memiliki margin yang rapi.
