Instruksi Tugas

Saatnya mempercantik halaman webmu dengan CSS!

Gunakan **Inline CSS** untuk mengubah warna teks elemen <h1> menjadi **biru** dan ukuran font menjadi **24px**.

Tambahkan blok <style> di dalam bagian <head> HTML-mu. Di dalamnya, atur warna latar belakang elemen <body> menjadi **#f0f0f0** (abu-abu terang) dan warna teks untuk semua elemen <p> menjadi **hijau**.
