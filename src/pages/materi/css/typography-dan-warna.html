Typography mengatur tampilan teks, termasuk ukuran, jenis font, dan gaya. Sedangkan warna digunakan untuk memberikan kesan visual yang kuat dan menambah estetika pada halaman web.

üìù Mengatur Typography

  • font-family: menentukan jenis huruf
  • font-size: menentukan ukuran teks
  • font-weight: menentukan ketebalan teks
  • text-align: menyelaraskan teks (left, center, right) 
p {
  font-family: Arial, sans-serif;
  font-size: 16px;
  font-weight: bold;
  text-align: center;
}

üé® Memberikan Warna

  • color: warna teks
  • background-color: warna latar belakang elemen
  • Warna dapat ditulis dalam format: nama warna (red), hex (#ff0000), rgb(rgb(255,0,0)) 
h1 {
  color: darkblue;
  background-color: #f0f0f0;
}

üí° Tips: Gunakan warna dan font yang serasi agar website kamu mudah dibaca dan enak dilihat!
