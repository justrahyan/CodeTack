üîé Selector Dasar (Class & ID)

Selector dalam CSS digunakan untuk memilih elemen HTML yang ingin diberikan style. Dua selector dasar yang paling sering digunakan adalah: Class dan ID.

üìò Selector Class (.)

Selector class menggunakan tanda titik (.) diikuti nama class. Ini digunakan untuk menargetkan satu atau beberapa elemen yang memiliki class tersebut. 

.kotak {
  background-color: lightblue;
  padding: 10px;
}
<div class="kotak">Saya punya class 'kotak'</div>

üÜî Selector ID (#)

Selector ID menggunakan tanda pagar (#) diikuti nama ID. ID bersifat unik dan hanya boleh digunakan satu kali di dalam halaman HTML. 

#judul {
  color: crimson;
  font-weight: bold;
}
<h1 id="judul">Ini Judul Utama</h1>

üí° Tips: Gunakan class untuk style berulang dan id untuk elemen yang unik.
