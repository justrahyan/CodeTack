🔎 Selector Dasar (Class & ID)

Selector dalam CSS digunakan untuk memilih elemen HTML yang ingin diberikan style. Dua selector dasar yang paling sering digunakan adalah: Class dan ID.

📘 Selector Class (.)

Class digunakan untuk elemen yang dapat dipakai berulang. Syntax diawali dengan titik (.) 

.kotak {
  background-color: lightblue;
  padding: 10px;
}
<div class="kotak">Saya punya class 'kotak'</div>

🆔 Selector ID (#)

ID digunakan untuk elemen unik. Syntax diawali dengan tanda pagar (#) 

#judul {
  color: crimson;
  font-weight: bold;
}
<h1 id="judul">Ini Judul Utama</h1>

💡 Tips: Gunakan class untuk elemen yang digunakan berulang, id hanya untuk elemen unik.
