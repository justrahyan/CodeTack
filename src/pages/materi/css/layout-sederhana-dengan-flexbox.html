Flexbox adalah metode modern di CSS untuk membuat layout yang fleksibel dan responsif. Dengan Flexbox, kita bisa menyusun elemen secara horizontal maupun vertikal dengan mudah, tanpa menggunakan float atau positioning manual.

üîß Properti Utama Flexbox

  • display: flex ‚Äì Mengaktifkan flex container
  • flex-direction ‚Äì Menentukan arah (row, column)
  • justify-content ‚Äì Mengatur posisi horizontal
  • align-items ‚Äì Mengatur posisi vertikal

üìÑ Contoh Kode

<style>
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.box {
  width: 100px;
  height: 100px;
  background-color: lightblue;
  text-align: center;
  line-height: 100px;
}
</style>

<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>

üí° Tips: Gunakan Flexbox saat kamu ingin membuat layout yang lebih adaptif dan mudah dikontrol, terutama untuk baris atau kolom.
