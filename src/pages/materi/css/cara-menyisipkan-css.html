🎨 Cara Menyisipkan CSS

CSS (Cascading Style Sheets) adalah aturan atau gaya yang digunakan untuk mempercantik tampilan halaman web. Dengan CSS, kita bisa mengubah warna, ukuran teks, jarak antar elemen, dan banyak lagi.

Tanpa CSS, halaman web akan terlihat polos seperti dokumen biasa. CSS memisahkan antara isi (teks, gambar) dan tampilan, sehingga desain halaman bisa diatur dengan lebih mudah dan konsisten.

Ada tiga cara untuk menyisipkan CSS ke dalam halaman HTML: Inline, Internal, dan External.

1️⃣ Inline CSS

Cara ini langsung menambahkan gaya (style) ke dalam elemen HTML. Cocok untuk perubahan cepat atau satu elemen saja. 

<p style="color: red;">Teks ini berwarna merah</p>

2️⃣ Internal CSS

CSS ditulis di dalam tag <style> di bagian atas (head) halaman HTML. Cocok untuk halaman tunggal yang memiliki banyak elemen dengan gaya yang seragam. 

<style>
p {
  color: blue;
}
</style>
<p>Teks ini berwarna biru</p>

3️⃣ External CSS

Cara ini menggunakan file CSS terpisah, lalu dihubungkan ke halaman HTML dengan tag <link>. Cara ini paling umum digunakan karena memudahkan pengaturan banyak halaman sekaligus. 

<link rel="stylesheet" href="style.css">
<p>Teks ini akan mengikuti aturan di file style.css</p>

⚙️ Coba Sendiri:

Kamu bisa mencoba menulis kode CSS langsung di bawah ini. Hasilnya akan muncul secara langsung di bagian pratinjau.
