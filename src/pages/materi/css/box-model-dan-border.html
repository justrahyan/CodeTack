Box Model & Border

Materi Belajar / CSS / Box Model & Border

üì¶ Box Model & Border

Dalam CSS, setiap elemen dianggap sebagai kotak yang terdiri dari beberapa bagian, yaitu: content, padding, border, dan margin. Ini disebut sebagai Box Model. Memahami box model sangat penting untuk mengatur layout dan jarak antar elemen.

üîç Bagian dari Box Model

  • content: Isi elemen, seperti teks atau gambar
  • padding: Ruang antara content dan border
  • border: Garis di sekitar elemen
  • margin: Jarak luar antara elemen dengan elemen lain

üß± Properti Border

  • border: Menentukan lebar, jenis, dan warna garis
  • border-radius: Membuat sudut elemen membulat
  • border-style, border-color, border-width untuk kontrol lebih spesifik 
.box {
  width: 200px;
  padding: 20px;
  margin: 20px;
  border: 2px solid #4ade80;
  border-radius: 10px;
  background-color: #f0fdf4;
}

üí° Tips: Gunakan box-sizing: border-box agar padding dan border tidak menambah total ukuran elemen.
